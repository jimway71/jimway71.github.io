关于Nginx

Nginx (pronounced “engine x”[7] /ˌɛndʒɪnˈɛks/ EN-jin-EKS) is a web server that can also be used as a reverse proxy, load balancer, mail proxy and HTTP cache. The software was created by Igor Sysoev and publicly released in 2004.[8] Nginx is free and open-source software, released under the terms of the 2-clause BSD license. A large fraction of web servers use Nginx,[9] often as a load balancer.[10] (From Wiki - Nginx)

Cloudflare免费SSL证书

参考:

  • Cloudflare启用通用SSL证书
  • 记录一次使用cloudflare生成15年的SSL证书

参照Cloudflare启用通用SSL证书的说明, 默认情况下，Cloudflare向所有Cloudflare域颁发并续订免费、非共享、公开信任的SSL证书。
Cloudflare提供了最长15年有效期的通用SSL证书, 用户可以在Cloudflare上为其拥有的由Cloudflare管理的域生成证书并使用。操作如下:

登录Cloudflare, 点击需要生成证书的域名后, 选择"SSL/TLS" - “源服务器”,

点击创建证书,

保持默认设置, 点击创建,

证书已经生成, 需要分别复制证书内容, 创建并保存对应的证书、密钥文件。

在Ubuntu(18.0.4)上配置Nginx

前述在Ubuntu上使用apt安装的Nginx, 配置文件在/etc/nginx, 主要的配置文件是nginx.conf以及其中include的相关配置文件

配置Nginx上的https服务器

nginx.conf中关于http项的配置中, 最后有两条include, 如下:

include /etc/nginx/conf.d/*.conf;
include /etc/nginx/sites-enabled/*;

其中/etc/nginx/sites-enabled/default文件中配置的http和https服务
修改/etc/nginx/sites-enabled/default, 屏蔽掉http服务(端口80), 增加https服务(端口443), 并配置ssl, 如下:
