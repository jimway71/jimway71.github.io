Microsoft Store 包 is a simple installation of Python that is suitable for running scripts and packages, and using IDLE or other development environments. It requires Windows 10 and above, but can be safely installed without corrupting other programs. It also provides many convenient commands for launching Python and its tools.

Pycharm开发环境

PyCharm是一个用于计算机编程的集成开发环境(IDE)，主要用于Python语言开发，由捷克公司JetBrains开发，提供代码分析、图形化调试器，集成测试器、集成版本控制系统，并支持使用Django进行网页开发。

  • PyCharm官网: https://www.jetbrains.com/pycharm/
  • pycharm如何配置python环境
  • Pycharm 外部工具设置

Qt图形框架模块PySide6

PySide6是来自于Qt for Python项目的官方Python模块，它提供了对完整Qt 6.0+框架的访问。

  • PySide6官网: https://pypi.org/project/PySide6/
  • 关于Qt开发: Qt for Python Deployment
  • 关于选用PySide, 参考: 调查五款Python GUI框架

编译打包工具

auto-py-to-exe

Auto PY to EXE是一个基于Eel 和PyInstaller构建的、通过简单的UI界面将python项目中的.py文件(可以是单个py文件或多个py文件)打包为.exe文件的简单工具，与PyInstaller相比，Auto PY to EXE更简单直观，可以自动生成执行转换代码，进而将python程序打包为exe文件。

  • 官网: https://pypi.org/project/auto-py-to-exe/
  • 简单说明: 图形化的PyInstaller (PyInstaller手册)
  • 使用参考: python打包教程

easycython

easycython.py是一个脚本，它将尝试自动将一个或多个.pyx 文件转换为相应的已编译.pyd|.so的二进制模块。
easycython.py is a script that will attempt to automatically convert one or more .pyx files into the corresponding compiled .pyd|.so binary modules。

  • 官网: https://pypi.org/project/easycython/

nuitka编译器

Nuitka是一个Python的替代编译器。 它可以无缝地替代和扩展Python的解释和编译工作。

  • Nuitka官网: https://nuitka.net/index.html
  • 参考文档:
    • Compare Nuitka VS Pyinstaller
    • 知乎 - Python打包exe的王炸-Nuitka
    • 知乎 - Nuitka之乾坤大挪移-让天下的Python都可以打包
    • PySide6学习笔记

在Windows系统中使用时:

  • 需要安装gcc编译器
    • 安装参考: 知乎 - Python打包exe的王炸-Nuitka
      下载MinGW-w64 - for 32 and 64 bit Windows Files中的x86_64-win32-sjlj(当前为GCC-8.1.0版本)
      下载完成后, 解压到选定目录下, 并在系统环境变量PATH中增加gcc.exe(在某目录/bin下)目录。
  • 当然, 还需要安装Nuitka
    • 安装参考: Nuitka Downloads 及 知乎 - Python打包exe的王炸-Nuitka
      使用以下命令进行安装:

      c:\> pip install -U “https://github.com/Nuitka/Nuitka/archive/develop.zip"
      # 用上面命令安装后, 编译时会警告, 并提示使用下面的命令(在报错信息及处理中有详细描述)
      c:\>pip install ordered-set Nuitka

在Windows下安装Python运行环境的完整安装过程(在sandbox上再操作了一遍)

在Windows系统上安装Python

下载当前最新的3.10版本, 选择Windows installer (64-bit)进行安装, 安装目录选择缺省目录, 安装最后选择"disable path length limit”。
安装完成后, 需要在系统环境变量path中增加python和python下scripts目录:

  • 在Windows搜索框中，键入"高级系统设置"，打开高级系统设置窗口;
  • 按下"环境变量"按钮，在打开的窗口中可以在"系统变量"框中选择Path，按"编辑"按钮;
  • 在(path环境变量)其中添加Python执行文件的目录, 在我的系统中, python.exe位于"C:\Users\WDAGUtilityAccount\AppData\Local\Programs\Python\Python310"目录下;
  • 并在(path环境变量)其中添加该目录下的scripts目录, 即"C:\Users\WDAGUtilityAccount\AppData\Local\Programs\Python\Python310\Scripts"目录 - pip等工具安装在这个目录。
  • 设置完成后, 就可以在命令行中执行python/pip等命令。

在Windows系统上安装Pycharm

从Download PyCharm下载Windows+Community版本。
当前版本为Pycharm-Community-2022.2.1。
安装使用的是当前可下载的Pycharm-Community-2022.2.1版本。
安装后的目录可以当绿色版本直接使用。

安装完成后, 需要做以下配置:

  • 配置外观
    在设置->外观与行为->外观里, 将主题改为"Windows 10 Light"方案。
  • 配置中文环境, 参考Pycharm 简单一步设置成官方中文版
    曾遇到的问题:
    在使用Pycharm-Community-2019.2版本(旧版)(原来下载的版本)的安装中, 在Plugins->Marketplace库中找不到"Chinese ​(Simplified)​ Language Pack / 中文语言包";
    手动从JetBrains官方插件网站下载的Chinese ​(Simplified)​ Language Pack / 中文语言包 For “Pycharm Community"插件, 在安装时报"不兼容"错误, 如下:

    改用当前最新版本2022.2.1后问题解决。
  • 配置Python环境, 参考pycharm如何配置python环境
    实际上, 在安装后, Pycharm的配置里已自动加入了Python解释器, 包括pip/setuptools/wheel。

在Windows系统上安装PySide6

  • 可以在"Window命令提示符"里安装PySide6, 也可以在PyCharm的"终端"里操作(建议), 命令很简单, 如下:

    pip3 install PySide6

    安装后, 在Python目录下的"Lib\site-packages\PySide6"目录中可以看到PySide6相关文件, 其中designer.exe可以直接运行(如果运行时报缺dll库错误, 只需要将所指dll文件拷到Pyside6目录下即可)

  • 在Pycharm中设置外部工具: Pyside6-Designer和Pyside6-UIC
    参考: 知乎 - Pycharm安装配置Pyside6

问题:

  • 本次安装后, 在Pycharm中试着运行Helloworld.py(需要PySide6支持)时, 报"ModuleNotFoundError: No module named ‘PySide6’“错误。
    本问题未确认原因, 也未找到解决方法。

在Windows系统上安装Nuitka

  • 安装gcc编译器
    安装参考: 知乎 - Python打包exe的王炸-Nuitka
    下载MinGW-w64 - for 32 and 64 bit Windows Files中的x86_64-win32-sjlj(当前为GCC-8.1.0版本)
    下载完成后, 解压到选定目录下, 并在系统环境变量PATH中增加gcc.exe(在某目录/bin下)目录。 完成后, 可以在cmd中使用以下命令以检查安装是否正常

    c:\> gcc --version

  • 安装Nuitka
    安装参考: Nuitka Downloads 及 知乎 - Python打包exe的王炸-Nuitka
    使用以下命令进行安装:

    c:\> pip install -U “https://github.com/Nuitka/Nuitka/archive/develop.zip"
    # 用上面命令安装后, 编译时会警告, 并提示使用下面的命令(在报错信息及处理中有详细描述)
    c:\>pip install ordered-set Nuitka

--onefile                   #打包成单个exe文件
