关于Cloudflare

Cloudflare是互联网上运行的最大网络之一。人们使用Cloudflare服务的目的是使其网站和服务更安全、更高效。
Cloudflare Docs网站上提供了相当丰富的文档资源, Cloudflare网站的"资源"引导也非常清晰。
Cloudflare社区是Cloudflare的官方社区。
Cloudflare Youtube是Cloudflare的Youtube帐号。

Cloudflare证书

参考:

  • Cloudflare免费通配符SSL/TLS证书最长15年申请教程

Cloudflare证书类型:

  • 源证书
    生成由 Cloudflare 签名的免费 TLS 证书，以安装在源服务器上。
    源证书仅对 Cloudflare 与源服务器之间的加密有效。
    可下载, 如用于https-web服务器, 在web浏览器会有安全警告。
  • 客户端证书
    使用客户端证书保护和验证您的 API 和 Web 应用程序。
    通过 mTLS 规则阻止不具备有效客户端 SSL/TLS 证书的设备发出的流量。
    可生成下载, 是给API和web应用使用的, 但没试过在https-web服务器使用
  • 边缘证书
    管理和购买将提供给您的 Web 访问者的 SSL 证书。
    这是Cloudflare 通用 SSL 证书，自定义主机名的是商业证书需要购买。
    免费套餐中不允许您上传任何 SSL 证书，但是可以订购自动续订证书或升级到 Business 计划来启用此功能。
    不可下载, 当然也没法用。

Cloudflared是Cloudflare提供的隧道客户端, 以前被称为"Cloudflare Argo Tunnel"。
Cloudflare免费开放了Cloudflared隧道客户端, 而且Cloudflared是Github上的开源项目。

Cloudflared隧道客户端的功能, 简单的说, 通过使用Cloudflared隧道客户端, 就是在任何可以访问Cloudflare的机器和Cloudflare中搭建一个安全的加密隧道，然后这个隧道可以用来承载很多不同种类型的流量，比如http(会被自动转换成 https)、tcp、ssh、smb、rdp, 甚至是unix套接字, 当前在Cloudflare相关文档中(文档一、文档二中, 明确说明以下协议可以被Cloudflared支持:

  • HTTP/S
  • HTTP over Unix socket
  • HTTPS over Unix socket
  • TCP
  • SSH
  • RDP
  • kubectl bastion mode
  • Hello World
  • HTTP status

通过使用Cloudflared隧道客户端,

  • 可以保证设备的安全(不需要开放更多端口以提供外部访问)
  • 可以利用Cloudflare网络在全球提高访问速度
  • 可以为通常无法在外部直接访问的设备, 如在NAT内部设备、仅有IPV4/IPV6的设备(而不能被仅有IPV6/IPV4的设备访问), 提供全球范围的访问。

简单的说, 就是:

  • 保护(设备)安全
  • 加快访问速度
  • 提供访问(通道)

注意:

  • 被访问设备(下面描述中为设备A)位于中国大陆时, 运行cloudflared tunnel run, 建立隧道时会报Err错误, 无法成功建立连接, 原因暂无法确认。

通过使用Cloudflared隧道客户端, 使IPV6设备的SSH端口可被IPV4设备访问

场景:

  • 被访问设备A: 位于公网上, 有 1*IPV6公网地址 + 1*IPV4内网地址
    可以通过IPV4/IPV6访问公网, 但仅可以通过IPV6地址被访问
  • 访问设备B: 位于内网中, 网络只有IPV4地址
    只能访问IPV4地址的设备
  • 目标: 设备B能访问设备A的SSH服务

  • cloudflare tunnel ssh

Cloudflare Spectrum

CloudflareSpectrum (收费服务)
Internet 不仅仅是网络。它包含许多其他与 Web 服务具有相同基本需求的 TCP / UDP 应用程序——都需要速度、安全性和可靠性。
Cloudflare Spectrum 是一种反向代理产品，可将 Cloudflare 的优势扩展到全部 TCP/UDP 应用程序。
**Cloudflare Spectrum 与 Argo Smart Routing集成，以比“尽最大努力”的互联网更快的速度发送 TCP 流量。**Cloudflare 的网络从数百万个互联网资产的流量中学习，从而实时绕过网络拥塞，实现基于机器学习（ML）的智能路由。
感觉是Cloudflared的加强版本。

Cloudflare WARP

Cloudflare WARP (基本版本免费, WARP+收费)。
Cloudflare的移动终端VPN, 从技术角度来看，WARP是一个VPN, 但它是为与传统VPN截然不同的用户设计的:

  • WARP不是用来让你在旅行时访问受地理限制的内容的。
  • 它不会在您访问的网站中隐藏您的IP地址。

如果您正在寻找这种高安全性的保护，那么传统的VPN或Tor之类的服务可能是更好的选择。
Cloudflare WARP的目的是:

  • 保护所有流量
  • 同时使其更快、更可靠

在安卓Play Store上的APP名称是"1.1.1.1", 搜索"WARP"也可以找到:

  • 1.1.1.1 + WARP: Safer Internet

但是, Unfortunately, 在中国大陆似乎无法正常使用这个APP。
