比较Streamlit、pywebio、Brython、remi这四种前段框架

框架名 官网 许可 特点 教程 资源
streamlit streamlit Apache 2.0 绘图功能挺强的，demo中就有一些画图的酷炫示例
很多的官方demo
每当你对Excel文件进行更改保存，Web页面还能够实时进行更新，确实挺不错的		 官方文档 各类资源
pywebio pywebio MIT 有丰富的官方示例
有很多demo		 PyWebIO官方使用手册中文版
Brython Brython BSD 3 有一些示例，但是并不酷炫
似乎是一个在JS中使用python语法的框架		 brython官方教程
remi remi Apache License 2.0 有一些网友写的教程
似乎与streamlit比较类似
有一个GUI的设计工具		 说明书在Github首页, 很简单

streamlit的文档很丰富(缺官方中文版, 有非官方中文版本-Streamlit API文档中文版 or Streamlit开发手册), 资源多, 功能强, 除上面所述功能外, 还能用于机器学习等功能, 在zhihu-streamlit中有不少介绍, 应当首先考虑选用。
